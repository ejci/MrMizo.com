#[MrMizo.com](http://mrmizo.com)
MrMizo.com web page



* Run `gulp` to start development server
* Run `gulp publish` to publish page on Amazon S3

For Amazon S3 you will need aws.json

```
{
   "key": "KEY",
   "secret": "SERET",
   "bucket": "BUCKET",
   "region": "REGION"
}
```
